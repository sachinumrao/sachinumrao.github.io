Using Virtual Environment in Python

While doing development sometimes we need to test our codes compatibility with different libraries. Such as when I create a machine learning model in Tensorflow, I verify if its working with cpu version and gpu version of library. Hence, I need to maintain two different versions of environments in python. Some people use "conda" but I am not much of a fan of conda and like to keep things as simple as they can be.


virutalenv is a python virtual environment manager. We can install it by following command.

pip install virtualenv
(OR)
sudo pip install virtualenv
