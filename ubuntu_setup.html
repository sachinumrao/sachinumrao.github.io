What I need in Ubuntu and How to install it

Basic Requirements

Update the system

sudo apt update
sudo apt upgrade
sudo apt dist-upgrade

Install basic packages

sudo apt install tlp tlp-rdw vim vlc gimp
sudo apt install ubuntu-restricted-extras
sudo apt install classicmenu-indicator

Developer Tools

Various libraries can be installed using following commands

sudo apt install gedit-common
sudo apt install python-pip python-dev
sudo apt install cmake build-essential
sudo pip install virtualenv
sudo pip install numpy scipy sklearn pandas matplotlib jupyter
sudo apt install git

Install Oracle Java

sudo apt-add-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer
